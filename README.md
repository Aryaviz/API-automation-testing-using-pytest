API automation testing using pytest
This is a comprehensive API automation testing framework for the Restful Booker API (https://restful-booker.herokuapp.com) built with pytest.
Reference for API links :BMayhew/awesome-sites-to-test-on: A curated list of sites to practice testing on 
Objectives
Verify authentication mechanism works correctly
Ensure proper error handling for invalid inputs
Validate GET operations for retrieving bookings
Validate PUT operations for updating bookings
Validate security through token-based authentication

API Environment
Base URL:  https://restful-booker.herokuapp.com
Protocol : HTTPS
Authentication : Token-based
Test Cases
TC1)  Create token with valid credentials
TC2) Create token with invalid credentials
TC3) Get all booking IDs
TC4) Get specific booking by ID
TC5) Update booking with valid token
